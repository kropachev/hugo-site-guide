# Сайт из репозитория GitHub за 30 минут. Hugo + Stack

Пошаговое руководство по созданию персонального сайта на [Hugo](https://gohugo.io/) с темой [Stack](https://github.com/CaiJimmy/hugo-theme-stack). Для разработчиков, которые хотят быстро получить красивый сайт без возни с фронтендом.  
- Сайт визитка, блог, или страница проекта.
- Бесплатно с хостингом на GitHub Pages
- Быстро и просто

## Что мы получим

Современный, адаптивный сайт с карточным дизайном, темной темой, поиском и RSS.

![Stack Theme Demo](https://github.com/CaiJimmy/hugo-theme-stack/raw/master/images/screenshot.png)

## Почему Hugo

Если вы уже используете GitHub - пишете код, выкладываете проекты или делаете заметки в README, то вы уже в шаге от полноценного сайта. Hugo выбран потому, что идеально вписывается в рабочий процесс разработчика: все, что требуется для сайта, остается в привычном виде - репозиторий, текстовые Markdown файлы и git-коммиты.

В отличие от CMS или облачных платформ, здесь сайт - это просто набор статических файлов. Никаких баз данных, серверных настроек или лишних зависимостей. Процесс сборки не сложнее, чем собрать release проекта: одна команда - и у вас готовый результат для публикации на GitHub Pages.

Markdown для контента не требует изучать HTML, достаточно того, как вы уже оформляете проекты на GitHub. А благодаря Git все под контролем - история изменений, откаты и автоматический деплой через Actions. Hugo оказался самым прямым и прозрачным способом сделать сайт для тех, кто привык работать с GitHub, не тратя время на настройку инфраструктуры.

## Почему тема Stack

[Stack](https://github.com/CaiJimmy/hugo-theme-stack) - одна из популярных тем Hugo (6000+ звезд на GitHub) с карточным дизайном, подходящим и для персональной страницы, и для технического блога.

- **Минимум настройки** Из коробки сайт выглядит современно и его не нужно допиливать или тратить время на кастомизацию. Вы быстро получите законченный сайт с хорошей навигацией.
- **Карточный дизайн и адаптивность.** Записи отображаются в виде информативных карточек с превью, и одинаково хорошо выглядят как на десктопе, так и в телефоне.
- **Темная тема.** Сайт автоматически подстраивается под темный режим пользователя или позволяет переключать тему вручную.
- **Поиск, оглавление, фильтрация.** Уже встроены быстрый поиск по всему контенту (работает прямо в браузере, без сервера), динамическое оглавление для длинных статей и сортировка по категориям и тегам. Не нужно ставить отдельные плагины.
- **Расширения для разработчиков.** Stack поддерживает встраивание кода, формулы через KaTeX, диаграммы Mermaid и галереи изображений.
- **Готовность к публикации.** Есть SEO-оптимизация, OpenGraph-метаданные для красивых превью в соцсетях, RSS и многое другое - все закрывает базовые сценарии разработчика и "из коробки".

В результате Stack - это идеальный выбор для тех, кто хочет минимальных хлопот с дизайном, современный интерфейс и расширенные фичи без погружения во фронт. Тема живет в GitHub, подходит под технические блоги, портфолио и сайт визитку разработчика.

## Что нам понадобится

- **GitHub аккаунт** - считаем что есть. Для хостинга сайта будет использоваться GitHub Pages.
- **Git** - тоже понятно.
- **Hugo Extended** - виновник торжества, для генерации статического сайта.
- **Go** - требуется Hugo для сборки сайта и работы с Hugo Modules (скачивание тем, расширений и генерация итоговых файлов)
- **Текстовый редактор** - например, VS Code.
